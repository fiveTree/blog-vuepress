#### git作用
Git版本管理工具的作用：
- 1、备份文件（类似U盘，网盘）；
- 2、记录历史（有各个阶段的代码历史版本）；
- 3、多端协作（可以在家工作，也可以在公司）；
- 4、团队协作（代码整合，多人共同完成一个项目）；
